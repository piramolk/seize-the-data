  * Plotting librairies
    * Matplotlib
    * Plotly
    * Seaborn
  * Scientific and Statistical Librairies

    * Pymc3
  * Machine Learning librairies
    * Scikit-learn
    * Tensorflow
  * Data manipulation
    * Pandas
    * Numpy
